# Ontology_Unity
Integration of an Ontology reasoner in Unity.

### Prerequisites

Unity Game Engine: https://unity3d.com/get-unity/download

* Install Unity
* Create a new Unity 3D project
* Import Ontology_Unity_Package.unitypackage
